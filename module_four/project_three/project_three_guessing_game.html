Project Three: Guessing Game

Project Four ❯

In this project, you’ll create a guessing game to help you practice generating different types of random numbers! You’ll also practice rounding using the Math utility class!

This game offers different difficulty levels to challenge guessing skills. The following table shows the difficulty levels and their meanings.

Difficulty Requirement
Easy Guess a random number between 1 and 10 in 15 tries
Medium Guess a random number between 1 and 100 in 10 tries
Hard Guess a random number between 1.0 and 100.0 in 20 tries
Impossible Guess a random number between 1.000 and 100.000 in 20 tries
