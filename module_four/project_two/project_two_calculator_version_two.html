Project Two: Calculator Version Two

Project Three ❯

The goal of this project is to make a better version of the calculator app you made in the previous project. You’ll practice using the Math utility class to perform basic mathematical operations, and you’ll program the calculator to accept text input like a real calculator! The calculator should support the following operations.

  • Addition.
  • Subtraction.
  • Multiplication.
  • Division.
  • Modulo.
  • Absolute value.
  • Square root.
  • Cube root.
  • Exponent.
  • Sine (both degrees and radians).
  • Cosine (both degrees and radians).
  • Tangent (both degrees and radians).

It should also support using the inputs “pi” and “e” to perform operations with π and e, respectively!
